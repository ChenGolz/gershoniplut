# סוניה ואמיר פלוט — עמודי זיכרון בעברית

החבילה מוכנה להעלאה ל־GitHub Pages.

## קבצים מרכזיים
- `index.html` — עמוד בחירת עיצוב
- `option-1-classic-book.html` — ספר זיכרון קלאסי ובהיר
- `option-2-modern-gallery.html` — עיצוב גלריה מודרני בגווני מרווה/חול
- `option-2-gallery.html` — גלריית תמונות מלאה עבור העיצוב השני
- `option-3-dark-museum.html` — מוזיאון זיכרון כהה ודרמטי
- `assets/shared/` — התמונות מהתיקייה שהועלתה
- `assets/docs/` — מסמכי הפרידה

## תיקונים בגרסה הזו
- הוסרו טעינות חיצוניות של פונטים כדי שהדפדפן לא ימשיך “לטעון לנצח”.
- הוגדרו גדלים קשיחים לתמונות בגלריות כדי שלא תופיע תמונה ענקית אחת.
- עמוד 2 קיבל צבעוניות דומה אך מובחנת, בגווני מרווה/כחול־ירקרק וחול.
- נוסף עמוד גלריה מלא לעמוד 2.
- תמונת הכיסוי בעמוד 3 הוזזה והוחשכה פחות, כדי שסוניה ואמיר יופיעו טוב יותר.

## העלאה ל־GitHub Pages
העלו את כל תוכן התיקייה לשורש הריפוזיטורי, ואז:
Settings → Pages → Deploy from branch → main / root


## עדכון טכני
הגרסה הזו כוללת CSS logical properties לתמיכת RTL יציבה יותר, רכיב Header משותף ב-JavaScript, אייקוני SVG נקיים במקום סימני טקסט, טיימליין Grid גמיש, ו-width/height על תמונות למניעת קפיצות טעינה.
